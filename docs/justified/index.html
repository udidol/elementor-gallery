Justified Gallery Demo

E-Gallery's Justified mode displays the gallery images in a customizable "horizontal masonry" layout. In this mode, the heights of all images are identical (except for the last row, the behavior of which is configurable), and the widths are variable so that the gallery images retain their original aspect ratio.
